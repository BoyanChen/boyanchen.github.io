Mr Bookmark

Mr Bookmark is a popup bookmarks. I'm personally interested in the origami. Especially the type of origami which is able to pop up by folding. We can see it variously used in some creative graphic design books.


Conceptualization

Instead of just make this interesting origami exist in the certain graphic design books. I came up with an idea to make it as a bookmarks. So that every times we fold the books we are going to read can have this kind of paper based sculpture pop up on the book!


Development

a. Prototypes

At first I decide the direction of folding bookmarks. Since it is too abstract to think about the perspective. I start to make some prototypes to help myself. Then I realized that I want to make the figure face directly to the users. So the book mark need to be fold start from the side of the book


b. Sketch

After Trying some origami figure, I tend to give the idea of making origami as a figure. Because origami itself is thick. If you fold it equally. It will get twice thicker. It is really inconvenient to fold for the users. So I start to try draw a figure by hand on a paper and cut it as a plat figure. I want the figure looks funny but not disturb users. Because every time they see it is the time they want to clam down and read book. So I just draw a very simple figure with just line and circle. And I call it Mr Bookmark.

Final

So the final output is like this. I also fold a handle at the edges of the bookmarks so that the users can open the book with the book mark easier.
